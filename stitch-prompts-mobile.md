# ElijoCreer — Prompts Stitch Mobile

## Prompt base de contexto

```text
Quiero diseñar una app web mobile-first llamada “ElijoCreer”, un Prode Mundial 2026 colaborativo y social donde usuarios crean cuenta, se unen a grupos, cargan pronósticos de partidos y compiten con amigos o familia.

La identidad visual debe estar inspirada en la Selección Argentina sin usar assets oficiales:
- estética moderna, premium, deportiva y social
- paleta principal celeste y blanco
- azul profundo para contraste
- acentos dorados sutiles inspirados en campeones del mundo
- sensación de comunidad, competencia amistosa, pasión futbolera y energía mundialista
- evitar look infantil, recargado o excesivamente gamer

Para mobile:
- priorizar navegación simple e intuitiva
- layouts compactos, limpios y escaneables con una sola mano
- cards modernas
- tabs claras
- CTA visibles
- excelente jerarquía visual para pronósticos, ranking y partidos
- experiencia tipo mezcla entre app social, fantasy sports y dashboard moderno

Voy a pedir las pantallas por tandas para mantener consistencia visual y calidad de UX.
```

---

## Tanda 1 — Onboarding y entrada

```text
Diseñá las interfaces mobile de onboarding y entrada principal para “ElijoCreer”, una app web tipo Prode Mundial 2026.

Contexto del producto:
- usuarios crean cuenta
- se unen a grupos de amigos, familia u oficina
- cargan pronósticos de partidos
- compiten con ranking de aciertos

Identidad visual:
- moderna, premium, deportiva y social
- inspirada en la Selección Argentina
- celeste y blanco como base
- azul profundo para contraste
- dorado sutil como acento

Necesito estas pantallas mobile-first:

1. Landing page pública
- hero emocional del Mundial 2026
- explicación breve de cómo funciona
- CTA crear cuenta
- CTA crear grupo o unirse a grupo
- beneficios: competir con amigos, ranking, reglas personalizadas y experiencia social
- preview visual de pronósticos, ranking y fixture

2. Inicio de sesión
- simple, moderno y confiable
- email y contraseña
- preparado visualmente para login social futuro

3. Registro
- flujo simple y amigable
- mensaje de valor del producto
- CTA posterior para crear grupo o unirse

4. Home autenticada inicial
- resumen de grupos del usuario
- próximos partidos
- pronósticos pendientes
- ranking rápido
- actividad reciente
- CTA principal para cargar pronóstico

Quiero una experiencia mobile muy clara, emocional, moderna y consistente.
```

---

## Tanda 2 — Juego social

```text
Diseñá las interfaces mobile centrales de “ElijoCreer”, enfocadas en la experiencia social y competitiva dentro de grupos.

Mantener esta identidad:
- estética moderna, premium, deportiva y social
- inspirada en la Selección Argentina
- paleta celeste, blanca, azul profundo y acentos dorados sutiles
- experiencia tipo app social + fantasy sports + dashboard moderno

Necesito estas pantallas mobile-first:

1. Crear grupo
- nombre
- descripción
- público o privado
- código o link de invitación

2. Pantalla principal del grupo
- encabezado del grupo
- ranking resumido
- miembros
- próximos partidos
- actividad reciente
- accesos rápidos a pronósticos, ranking y configuración

3. Fixture del Mundial 2026
- partidos por fase y fecha
- filtros simples
- estados claros: próximo, en juego, finalizado

4. Carga de pronóstico
- equipos, escudos, fecha y hora
- inputs simples para goles
- estado visual si aún se puede editar o si ya cerró
- experiencia ultra rápida desde mobile

5. Resultados y scoring
- resultado oficial
- comparación contra el pronóstico del usuario
- puntos obtenidos
- feedback visual claro: exacto, ganador, falló

6. Ranking del grupo
- tabla moderna mobile-friendly
- avatar, nombre, puntos, aciertos y tendencia
- top 3 destacado

Busco una experiencia ágil, clara, social y muy futbolera.
```

## Tanda 3 — Configuración y administración

```text
Diseñá las interfaces mobile de configuración, perfil y administración para “ElijoCreer”.

Contexto del producto:
- cada grupo puede personalizar sus reglas de puntaje
- hay grupos públicos o privados
- existe un admin de grupo y un admin global del sistema

Mantener identidad visual:
- moderna, premium, deportiva y social
- inspiración Selección Argentina
- celeste, blanco, azul profundo y dorado sutil

Necesito estas pantallas mobile-first:

1. Configuración de reglas del grupo
- editable por admin del grupo
- puntos por resultado exacto
- puntos por acertar ganador o empate
- bonus opcionales
- explicación simple de cómo se calcula el puntaje

2. Perfil del usuario
- avatar
- estadísticas personales
- grupos en los que participa
- porcentaje de aciertos
- historial de pronósticos

3. Administración del grupo
- gestión de miembros
- tipo de grupo público o privado
- invitaciones o link para compartir
- permisos y accesos

4. Panel admin global
- carga de resultados oficiales
- gestión de partidos
- control del fixture del Mundial 2026
- diseño claro y operativo, pero consistente con la app

Quiero claridad, control y confianza, con layouts compactos y muy usables en mobile.
```
