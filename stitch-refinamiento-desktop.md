# ElijoCreer — Prompt de Refinamiento Stitch Desktop

```text
Quiero refinar las interfaces desktop ya generadas para la app web “ElijoCreer”.

Importante:
- NO quiero rediseñar el producto desde cero
- quiero mantener lo mejor de las pantallas actuales
- quiero corregir inconsistencias visuales, funcionales y de UX
- usar como fuente de verdad el DESIGN.md y el backlog funcional del MVP

Contexto del producto:
ElijoCreer es una app colaborativa y social de pronósticos para el Mundial 2026. Los usuarios crean cuenta, se unen a grupos públicos o privados, cargan pronósticos, compiten con amigos y ven rankings por grupo. Cada grupo puede tener reglas de puntaje propias configuradas por su admin. Los resultados oficiales son globales para todos los grupos.

Objetivo del refinamiento:
- mejorar consistencia visual
- alinear la UX a reglas reales del negocio
- reforzar jerarquía, estados, acciones y permisos
- aprovechar mejor el espacio desktop
- mantener una identidad moderna inspirada en la Selección Argentina

Tomar como base visual estas reglas:
- estética moderna, premium, deportiva y social
- paleta celeste, blanco, azul profundo y dorado sutil
- look limpio, emocional, claro y elegante
- evitar estilo infantil, recargado o demasiado gamer
- desktop con grids, paneles y tablas claras
- experiencia tipo mezcla entre app social, fantasy sports y dashboard moderno

Tomar como base funcional estas reglas:
- el torneo del MVP es solo Mundial 2026
- un usuario puede pertenecer a varios grupos
- los grupos pueden ser públicos o privados
- el admin del grupo configura las reglas de puntaje
- el usuario puede editar su pronóstico solo antes de que empiece el partido
- al comenzar el partido, el pronóstico queda cerrado
- el resultado oficial lo carga un admin global
- el ranking es independiente por grupo

Refinar estas pantallas desktop:

1. Home autenticada
Quiero que muestre con más claridad:
- grupos del usuario
- próximos partidos
- pronósticos pendientes
- CTA principal para cargar pronóstico
- acceso rápido al ranking
- actividad reciente simple

2. Pantalla de grupo
Quiero que se entienda mejor:
- nombre e identidad del grupo
- miembros
- ranking resumido
- próximos partidos
- accesos rápidos a pronósticos, ranking y configuración
- si el grupo es público o privado

3. Fixture del Mundial 2026
Quiero estados inequívocos:
- próximo
- en juego
- finalizado
También quiero una vista desktop con filtros claros por fase o fecha y mejor lectura analítica.

4. Carga de pronóstico
Quiero una experiencia muy clara, rápida y ordenada.
Debe mostrar:
- escudos
- equipos
- fecha y hora
- inputs claros de goles
- estado visible: editable o cerrado
- feedback claro al guardar

5. Resultados y scoring
Quiero una comparación visual fuerte entre:
- resultado oficial
- pronóstico del usuario
- puntos obtenidos
- motivo del puntaje: exacto, acertó ganador/empate, sin acierto

6. Ranking del grupo
Quiero una tabla desktop más rica con:
- posición
- avatar
- nombre
- puntos
- aciertos
- tendencia
- top 3 destacado

7. Reglas del grupo
Quiero que la pantalla deje muy claro:
- que las reglas son definidas por el admin
- cuánto vale acertar exacto
- cuánto vale acertar ganador o empate
- si hay bonus
- explicación simple de cómo se calcula el puntaje

8. Administración del grupo
Quiero una vista clara para admin con:
- miembros
- invitación por link o código
- visibilidad pública o privada
- accesos a configuración

9. Perfil del usuario
Quiero mostrar:
- nombre y avatar
- grupos en los que participa
- estadísticas básicas
- porcentaje de aciertos
- historial resumido

10. Panel admin global
Quiero una interfaz operativa, clara y consistente para:
- cargar resultados oficiales
- gestionar partidos
- visualizar fixture del Mundial 2026

Lineamientos de refinamiento UX:
- priorizar información realmente útil
- reducir ruido visual
- reforzar jerarquías y lectura de datos
- hacer que los estados sean visualmente obvios
- reforzar CTAs principales
- mejorar claridad operativa para rankings, partidos y administración

Lineamientos de refinamiento visual:
- unificar cards, badges, botones, espaciados y tipografía
- mejorar jerarquía de titulares, tablas, métricas, estados y acciones
- destacar resultados, puntajes y ranking sin recargar
- hacer que la marca ElijoCreer se sienta emocional, futbolera y moderna

No inventar funcionalidades fuera del MVP.
Refinar en base al producto real y a las pantallas ya generadas.
```
